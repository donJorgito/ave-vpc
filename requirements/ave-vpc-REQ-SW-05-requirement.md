### ave-vpc.REQ-SW-05 - Git 2.39 o superior

**Description:**

Git se utiliza para clonar el repositorio del proyecto y para clonar
el código fuente de mlvpn durante la compilación. Versión mínima:
2.39 (incluido en Xcode CLT 14+).

**Parent Requirement:** ave-vpc.REQ-SW-02

**Acceptance Criteria:**

- `git --version` devuelve `2.39.x` o superior.
